# inventory-system

Aplikasi Inventory System ini dibuat dengan menggunakan Express.js untuk backend dan Flutter untuk frontend. Aplikasi ini menyediakan fitur untuk mengelola inventaris barang, seperti menambah, mengurangi, memperbarui, dan menghapus data barang.

Fitur

Login: Pengguna dapat melakukan login untuk mengakses aplikasi.

Tambah Barang: Menambahkan barang baru ke dalam sistem inventaris.

Kurang Barang: Mengurangi jumlah barang yang ada dalam inventaris.

Update Barang: Memperbarui informasi barang yang sudah ada.

Hapus Barang: Menghapus barang dari inventaris.

Teknologi yang Digunakan

Backend:

Express.js (Node.js framework)

Posgrees (Database)

Frontend:

Flutter (UI Framework)
